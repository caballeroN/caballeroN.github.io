Calculadora del IMC para adultos (SMD)

(Sistema métrico decimal | Sistema inglés)

Bienvenido
Esta calculadora proporciona el IMC y la correspondiente categoría de nivel de peso según el IMC. Puede calcular el IMC en el sistema métrico decimal (SMD), o el sistema inglés (ENG). Utilice esta calculadora para mayores de 20. Para menores de 20, utilice la calculadora de IMC para adolescentes y niños.
